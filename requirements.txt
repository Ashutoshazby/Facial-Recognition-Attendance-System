How to use:
Make sure your virtual environment is activated:

bash
Copy code
.venv\Scripts\activate   (Windows)
source .venv/bin/activate  (Linux/macOS)
Install all dependencies in one command:

nginx
Copy code
pip install -r requirements.txt
This ensures OpenCV, Dash, and Pandas are installed in your environment.

After this, you can run both your attendance system and dashboard without errors.
